jar包其实压缩包，打开以后是一个文件夹。

## fabric

如何获取模组名称和版本

压缩包打开后有一个`fabric.mod.json`文件，里面有

```json
{  
    "id": "fabric-api",
  	"name": "Fabric API",
  	"version": "0.147.0+26.2",
    "icon": "sodium-icon.png",
}
```



## forge

压缩包打开后，在`META-INF`文件夹下有个`mods.toml`

```toml
[[mods]] #mandatory
modId = "xaerominimap" #mandatory
version = "25.3.12" #mandatory
displayName = "Xaero's Minimap" #mandatory
```



## neoforge

压缩包打开后，在`META-INF`文件夹下有个`neoforge.mods.toml`

```toml
[[mods]] #mandatory
modId = "xaerominimap" #mandatory
version = "25.3.14" #mandatory
displayName = "Xaero's Minimap" #mandatory
logoFile = "压缩包内路径"
```
